# advanced_programming_exam
Exam repository for Advanced Programming course.  

Authors: Lorenzo Basile, Arianna Tasciotti.

## Structure of the repository

- Folder `doxygen`: this folder contains file `doxy.in`, Doxygen configuration file and `documentation.pdf`, the documentation pdf file generated by Doxygen.
- Folder `benchmark`: this folder contains C++ source file `benchmark.cpp` used to measure the performances of our container and `std::map`, the Python script `script.py` used to run the program automatically and to process results, `Makefile` used to produce the executable `benchmark.x`.
- `bst.hpp`: header file containing class `bst`.
- `node.hpp`: header file containing class `node`.
- `iterator.hpp`: header file containing class `_iterator`.
- `main.cpp`: source file used to test functions of `bst`.
- `Makefile`: makefile that produces the executable `main.x`.
- `report.pdf`: report about the implementation of our container `bst` and results of benchmarking.
